🛡️ Cyber Deception Honeypot Project
-
This project is a Cyber Deception Honeypot built using Python.
It simulates a vulnerable login service to attract, detect, and log fake cyber-attacks in a safe and controlled environment.

The goal of this project is to demonstrate cybersecurity concepts such as honeypots, brute-force attacks, logging, and monitoring — without performing real hacking.

-->Objectives

Understand honeypot and deception technology

Simulate brute-force and weak credential attacks

Capture and store attack logs

Display attack activity using a live web dashboard

Provide a safe, educational cybersecurity demo

--> Technologies Used

Python 3

Socket Programming

Flask (Web Framework)

HTML / CSS

JSON (Log Storage)

Windows Batch Script (One-click execution)
 
--> How to Run the Project
Step 1: Install Python

Make sure Python 3 is installed.
Check using:

python --version

Step 2: Install Required Library
pip install flask

Step 3: Run the Project (One Click)

Just double-click:

start_project.bat

This will:

Start the honeypot

Simulate attacks

Open the live dashboard

Step 4: View Live Logs

Open browser:

http://127.0.0.1:5000


You will see:

Timestamp

IP address

Attack type

Severity

Username & password used

--> Main Features

Fake login honeypot

Randomized attack simulation

Severity classification (INFO, WARNING, CRITICAL)

JSON-based structured logging

Live web dashboard

One-click execution
